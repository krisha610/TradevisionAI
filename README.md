# 📈 TradeVision AI — Recurrent Neural Engine

[![Streamlit App](https://static.streamlit.io/badge_streamlit.svg)](https://tradevision-ai.streamlit.app)
[![GitHub License](https://img.shields.io/github/license/krisha610/TradevisionAI)](https://github.com/krisha610/TradevisionAI/blob/main/LICENSE)

**TradeVision AI** is a professional-grade stock analysis and prediction platform powered by Recurrent Neural Networks (RNN). It combines deep learning forecasting with a high-performance technical analysis dashboard and real-time market sentiment.

🚀 **Live Site**: [tradevision-ai.streamlit.app](https://tradevision-ai.streamlit.app)

---

## ⚡ Core Features

- **🤖 AI Forecasting Engine**: Predict tomorrow's closing price with deep learning (SimpleRNN, LSTM, GRU).
- **📊 Interactive Charts**: Custom-themed Plotly charts for Candlesticks, RSI, Bollinger Bands, and Volume.
- **📡 Real-Time Price Feed**: Live data streaming from global exchanges (NSE, BSE, NYSE, NASDAQ).
- **🔔 Smart Alerts**: Automated price monitoring and email notifications (SMTP-integrated).
- **💼 Portfolio Tracker**: Persistent dashboard to monitor your holdings and P&L.
- **📰 Sentiment Analysis**: AI-powered keyword scoring from global news headlines.

---

## 🛠️ Tech Stack

- **Frontend/UI**: [Streamlit](https://streamlit.io/) (Python)
- **Deep Learning**: [TensorFlow](https://www.tensorflow.org/) & [Scikit-Learn](https://scikit-learn.org/)
- **Data Source**: [yfinance](https://github.com/ranaroussi/yfinance)
- **Visuals**: [Plotly](https://plotly.com/python/)
- **Storage**: SQLite (Persistent SQL Database)

## 🚀 Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/krisha610/TradevisionAI.git
   cd TradevisionAI
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv tv_env
   source tv_env/bin/activate  # On Windows: tv_env\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Add Secrets** (Optional for local use):
   Create a `.streamlit/secrets.toml` with your Gmail credentials for email alerts:
   ```toml
   GMAIL_USER = "your-email@gmail.com"
   GMAIL_APP_PASSWORD = "xxxx xxxx xxxx xxxx"
   ANTHROPIC_API_KEY = "sk-..."
   ```

5. **Run the app**:
   ```bash
   streamlit run app.py
   ```

## 🌍 Deployment

This app is optimized for **Streamlit Community Cloud**. To deploy your own:

1. Push your code to GitHub.
2. Link your repository at [share.streamlit.io](https://share.streamlit.io).
3. Add your secrets to the "Advanced Settings" on the dashboard.
